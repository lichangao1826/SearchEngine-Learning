## 网络爬虫 & 网页去重

搜索引擎的信息源来自于互联网网页，通过网络爬虫将整个互联网的信息获取到本地，因为互联网页面中有相当大比例的内容是完全相同或者近似重复的，“网页去重”模块会对此做出检测，并去除重复内容。

*具体可参考：[网络爬虫](./网络爬虫) 和 [网页去重](./网页去重)*
